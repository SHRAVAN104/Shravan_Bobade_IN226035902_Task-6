# Shravan_Bobade_IN226035902_Task-6

This project contains solutions to six real-world problem statements implemented using Python functions.
Each problem demonstrates structured programming, conditional logic, loops, and data structure usage.

## **Problems Implemented**
1️] Smart Parking Lot Management System

-Tracks vehicle entry and exit logs 

-Calculates currently parked vehicles

-Identifies peak parking usage

-Alerts if capacity is exceeded

2️] Online Food Delivery Time Estimator

-Estimates delivery time based on:

  Distance ,
  Traffic level ,
  Weather condition 
  
-Applies dynamic delay adjustments

3️] Movie Theatre Seat Occupancy Analyzer

-Calculates occupancy percentage

-Determines show status (Available / Almost Full / Housefull)

4️] Cloud Server Load Classification System

-Computes average CPU usage

-Classifies server status:
  Normal ,
  Warning ,
  Critical

5️] Smart Classroom Resource Usage Monitor

-Tracks usage hours of resources

-Identifies overused resources

-Triggers energy alerts

6️] Online Event Registration Capacity Controller

-Tracks event registrations

-Prevents overbooking

-Triggers waitlist mode when capacity is exceeded
